# Research Seed Material

These documents summarize findings from three research passes conducted against a v1 work harness. They are **seed material** — background knowledge that informs but does not constrain the v2 design.

Every conclusion below was reached by agents operating inside v1's context. The anchoring research (see findings below) confirmed that agents in that context tend to extend existing patterns rather than question them. Read these findings critically.

## Documents

1. `rewrite-assessment.md` — Internal audit of v1: complexity, degradation patterns, path dependency
2. `paradigm-shift.md` — External landscape: what the ecosystem looks like in March 2026
3. `landscape-summary.md` — Prior landscape research: frameworks, MCP, multi-agent patterns
