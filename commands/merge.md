# /furrow:merge

Reserved command design. This slash command and `furrow merge` are not
implemented in the Go CLI.

This document is retained only as historical planning material for a possible
future merge workflow. It is not a live operator procedure, canonical command
surface, or implementation contract.

Prior shell-based merge planning notes described compatibility wrappers and
phase scripts that remain outside the Go command surface. Where those notes
refer to transitional shell behavior, treat it as a temporary compatibility
holdout, not as the behavior of `/furrow:merge` or `furrow merge`.
